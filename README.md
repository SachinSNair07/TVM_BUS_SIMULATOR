# 🚌 TVM Bus Simulator

A highly realistic bus driving simulation game set in Thiruvananthapuram (Kerala), focused on authentic vehicle physics, real-world routes, and immersive gameplay systems.

---

## 🎯 Project Overview

**TVM Bus Simulator** is an indie game development project built using Unreal Engine 4, aiming to replicate real-life bus driving conditions in Kerala.

Unlike arcade-style driving games, this project emphasizes:
- Realistic vehicle behavior
- Accurate environmental scale
- Authentic gameplay systems (passengers, conductor, routes)

This project is being developed as a solo effort with a focus on performance optimization for both mobile and PC platforms.

---

## 🌍 Core Vision

To create a **Kerala-based simulation experience** that feels authentic, grounded, and technically solid, while remaining accessible on low-end devices.

---

## 🔥 Key Features

### 🚗 Vehicle System
- Realistic bus driving mechanics using UE4 Vehicle System
- Proper wheel setup (radius, suspension, physics)
- Steering system with smooth response
- Planned gearbox and engine behavior system

---

### 🚌 Bus Model
- Custom-built 3D tourist bus created in Blender
- Optimized for real-time rendering
- Designed with correct real-world scale
- Modular structure for future upgrades

---

### 🛣️ Real-World Route System
- Inspired by real locations in Thiruvananthapuram
- Initial route:
  - Vizhinjam Bus Stand → East Fort
- Future expansion planned with multiple routes

---

### 👥 NPC System (Work in Progress)
- Passengers entering and exiting bus
- Conductor system (movement + ticket handling)
- Seat interaction system

---

### 🌦️ Environment System
- Day/Night cycle
- Dynamic weather (rain system planned)
- Lighting system optimized for performance

---

### 🚪 Interactive Systems
- Door animations (open/close)
- Planned:
  - Indicator lights
  - Brake lights
  - Headlights
  - Reverse lights

---

### 📱 Optimization
- Targeting **low-end Android devices**
- Reduced draw calls and optimized assets
- Efficient lighting setup (mix of dynamic and static)
